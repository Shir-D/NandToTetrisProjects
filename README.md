# From Nand to Tetris: Building a Modern Computer from First Principles

## Project Overview:
A comprehensive project that involves building a complete computer system, starting from the most basic logic gate (Nand) and scaling up to a fully functional hardware platform capable of executing machine code. This project demonstrates an understanding of how hardware and software layers interact to create a working computer.


## Key Technical Achievements:

* **Hardware Construction:**
  Implemented fundamental logic gates (AND, OR, XOR), a 16-bit Adder, and a fully functional Arithmetic Logic Unit (ALU) using a Hardware Description Language (HDL).
 
* **CPU & Memory Design:**
  Designed and built the central processing unit (CPU), instruction decoding logic, and memory systems (RAM/ROM) to support program execution.
 
* **Low-Level Software Stack:**
  Developed an Assembler in Java to translate symbolic assembly language into binary machine code, bridging the gap between hardware and software.

* **Systems Thinking:**
  Gained hands-on experience with computer architecture, fetch-execute cycles, and the internal workings of a computer at the gate level.
